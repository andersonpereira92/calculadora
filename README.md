# Calculadora Kivy

Uma calculadora simples feita com Kivy.

## Instalação

1. Crie e ative um ambiente virtual:

    ```bash
    python -m venv venv
    venv\Scripts\activate
    ```

2. Instale as dependências:

    ```bash
    pip install -r requirements.txt
    ```

## Uso

Execute o script principal:

```bash
python src/calculadora.py
